# defispartan.xyz

Home for defispartan, built with React + Next + TypeScript + Bun.

## Install

This project uses Bun as the default package manager, visit [bun.sh](https://bun.sh/) to get started.

`bun install`

## Run

`bun run dev`

## Build

`bun run build`

## Serve Build Preview

`bun run preview`
